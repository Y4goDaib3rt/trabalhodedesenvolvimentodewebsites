Descrição do Projeto:

O projeto "Portfólio Yago Daibert" foi desenvolvido como parte do trabalho de desenvolvimento de websites para a disciplina de Web Design da Faculdade Metodista Granbery. O objetivo principal é criar um site personalizado que sirva como uma plataforma para apresentar suas habilidades, projetos e conquistas profissionais de forma organizada e atrativa.



Tecnologias usadas:

-Nome: HTML (HyperText Markup Language).

-Site Oficial: https://www.w3.org/html/.

-Definição da tecnologia: O HTML é a linguagem de marcação padrão para a criação e estruturação de páginas web. Ele fornece a estrutura básica de uma página, definindo os elementos e tags que compõem a estrutura do conteúdo. O HTML utiliza uma sintaxe de marcação para indicar como o conteúdo deve ser exibido em um navegador web.

-Prós:

Simplicidade: O HTML é uma linguagem fácil de aprender e usar, com uma sintaxe simples e direta.

Compatibilidade: As páginas HTML são suportadas por todos os principais navegadores web, tornando-as acessíveis em várias plataformas.

Flexibilidade: O HTML permite a criação de uma ampla variedade de elementos e estruturas para representar diferentes tipos de conteúdo na web.

-Limitações:

Limitado à estruturação: O HTML é focado principalmente na estruturação do conteúdo e não oferece recursos avançados de estilização e interatividade.

Falta de formatação semântica: O HTML por si só não fornece uma formatação semântica completa, sendo necessário usar CSS para estilizar a aparência da página.

Dificuldade em criar layouts complexos: Para criar layouts complexos e personalizados, pode ser necessário utilizar técnicas avançadas de CSS e outras tecnologias complementares.

-Grandes projetos que utilizam o HTML:

>Wikipedia: A enciclopédia online utiliza HTML para estruturar seu conteúdo e apresentá-lo de forma organizada.

>Amazon: O gigante do comércio eletrônico utiliza HTML para construir suas páginas de produtos, carrinho de compras e checkout.

>Google: O mecanismo de busca mais popular do mundo utiliza HTML para exibir os resultados da pesquisa e fornecer uma interface amigável aos usuários.

>Facebook: A rede social utiliza HTML para construir o layout de suas páginas, incluindo a exibição de posts, comentários e perfis de usuário.

>Twitter: A plataforma de mídia social utiliza HTML para renderizar tweets, perfis de usuário e exibir a linha do tempo dos usuários.

>GitHub: A plataforma de hospedagem de código utiliza HTML para exibir repositórios, código-fonte e informações relacionadas a projetos.

CSS

-Nome: CSS (Cascading Style Sheets).

-Site Oficial: https://www.w3.org/Style/CSS/.

-Definição da tecnologia: O CSS é uma linguagem de estilo utilizada para definir a aparência e o layout de elementos em uma página web. Ele complementa o HTML, permitindo que os desenvolvedores especifiquem cores, fontes, margens, espaçamentos, posicionamentos e outros aspectos visuais dos elementos HTML.

-Prós:

Flexibilidade: O CSS permite uma personalização completa da aparência de uma página web, separando a camada de estilo da estrutura do conteúdo.

Reutilização: É possível criar classes e estilos reutilizáveis, o que facilita a manutenção e a consistência visual em todo o site.

Responsividade: O CSS oferece recursos para criar layouts responsivos, permitindo que as páginas se adaptem a diferentes tamanhos de tela e dispositivos.

-Limitações:

Curva de aprendizado: O CSS pode ser complexo para iniciantes, requerendo um tempo de aprendizado para dominar os conceitos e as melhores práticas.

Inconsistências entre navegadores: Alguns recursos avançados do CSS podem ser interpretados de maneira inconsistente pelos diferentes navegadores, exigindo ajustes adicionais.

Limitações de layout: O CSS ainda tem algumas limitações em relação ao design de layouts complexos, especialmente em comparação com ferramentas de design gráfico dedicadas.

-Grandes projetos que utilizam o CSS:

>YouTube: O site de compartilhamento de vídeos utiliza CSS para estilizar a interface do usuário e criar uma experiência visual atraente.

>Airbnb: A plataforma de hospedagem online utiliza CSS para definir o layout, as cores e os estilos de elementos em suas páginas.

>LinkedIn: A rede social profissional utiliza CSS para estilizar perfis de usuário, listagens de empregos e outros componentes da plataforma.

>The New York Times: O renomado jornal utiliza CSS para criar layouts responsivos, tornando o conteúdo acessível em diferentes dispositivos.

>Slack: A plataforma de comunicação empresarial utiliza CSS para estilizar as conversas, menus e outros elementos da interface do usuário.

>Pinterest: A rede social de compartilhamento de imagens utiliza CSS para criar grades e estilizar o conteúdo exibido em seus painéis.

JAVASCRIPT

-Nome: JavaScript.

-Site Oficial:https://developer.mozilla.org/en-US/docs/Web/JavaScript.

-Definição da tecnologia: O JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. Ela é amplamente utilizada para adicionar interatividade, dinamismo e funcionalidades avançadas às páginas web. O JavaScript permite a manipulação de elementos HTML, o controle de eventos, a interação com servidores, a criação de animações e muito mais.

-Prós:

Interatividade: O JavaScript permite adicionar interações em tempo real às páginas web, como validação de formulários, exibição de conteúdo dinâmico e criação de elementos interativos.

Versatilidade: O JavaScript pode ser executado em praticamente qualquer navegador web, bem como em servidores, o que o torna uma tecnologia versátil para desenvolvimento web full-stack.

Vasta comunidade e recursos: O JavaScript possui uma comunidade ativa e uma ampla gama de bibliotecas e frameworks disponíveis, facilitando o desenvolvimento de projetos e a resolução de problemas.

-Limitações:

Segurança: Como o código JavaScript é executado no navegador do cliente, é importante ter cuidado com a segurança, pois o código pode ser acessado e modificado por usuários mal-intencionados.

Dependência do ambiente do cliente: A execução do JavaScript depende da disponibilidade de um ambiente do cliente, como um navegador web, o que pode limitar sua utilização em outros contextos.

Desempenho: Em alguns casos, o JavaScript pode ter um desempenho inferior em comparação com outras linguagens de programação devido ao fato de ser interpretado e executado em tempo de execução.

-Grandes projetos que utilizam o CSS:

>Facebook: A plataforma do Facebook utiliza JavaScript extensivamente para fornecer uma experiência interativa e em tempo real aos usuários.

>Google Maps: O serviço de mapas online do Google utiliza JavaScript para fornecer recursos de mapeamento interativo e navegação.

>Netflix: O popular serviço de streaming de vídeos utiliza JavaScript para criar uma interface de usuário dinâmica e personalizada para seus usuários.

>Spotify: A plataforma de streaming de música utiliza JavaScript para reprodução de áudio, criação de playlists e recomendações personalizadas.

>Trello: A ferramenta de gerenciamento de projetos utiliza JavaScript para permitir a criação, organização e colaboração de tarefas em tempo real.

>Airbnb: O site de hospedagem online utiliza JavaScript para fornecer uma experiência de reserva e pesquisa de propriedades interativa e responsiva.

Descrição do Projeto:

Neste projeto de vídeo, exploraremos a vida e o trabalho do renomado pintor Frans Hals. Com foco em seu estilo único, influência artística e contribuições para a pintura do século XVII, o vídeo tem como objetivo educar e inspirar os espectadores a descobrirem a beleza e a importância do legado deixado por esse grande mestre holandês.

O vídeo mergulha na história de Frans Hals, desde seus primeiros anos até seu impacto duradouro no mundo da arte. Por meio de uma narração envolvente e de imagens cuidadosamente selecionadas, o vídeo revela a vida pessoal de Hals, suas influências artísticas e o contexto cultural em que ele viveu.
